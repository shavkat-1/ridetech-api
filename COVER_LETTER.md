# Сопроводительное письмо

Здравствуйте!

Выполнил тестовое задание по разработке RESTful API для сервиса RideTech.

Что реализовано:

* Регистрация, авторизация и выход пользователей через Laravel Sanctum.
* Роли пассажира и водителя.
* Создание, просмотр, обновление и отмена поездок.
* Просмотр доступных поездок для водителей.
* Принятие и завершение поездок.
* Управление транспортными средствами водителя.
* Система отзывов и рейтингов.
* Пагинация и фильтрация поездок.
* Авторизация через Laravel Policies.
* Swagger-документация и Postman Collection.
* Feature-тесты для основных бизнес-сценариев.

Дополнительно при разработке использовал многослойную архитектуру Controller → Service → Repository, что позволило отделить бизнес-логику от слоя контроллеров и работы с данными.

Проект реализован как MVP согласно требованиям задания  и гоьов к проверке.
